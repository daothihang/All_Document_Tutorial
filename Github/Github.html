HƯỚNG DẪN CÀI ĐẶT VÀ SỬ DỤNG GITHUB

I Hướng dẫn cài đặt

Bước 1: Truy cập vào đường link sau để download github:

        https://git-scm.com/downloads

Note: kiểm tra máy tính sử dụng window 32 hay 64 bit để tải bản cho phù hợp.

Bước 2: Cài đặt rất đơn giản khi download xong bạn click vào file tải về và cài đặt (nó rất đơn giản chỉ next)

OK, Bạn vừa cài đặt xong Git.

Sau khi cài đặt xong, nhấn phải chuột vào một thư mục bất kỳ, một Context-Menu sẽ hiển thị, bạn có thể nhìn thấy các Menu-Item của Git, điều này chứng tỏ rằng bạn đã cài đặt Git thành công.

II Hướng dẫn sử dụng github

  • Để làm việc với github ta cần:

Đăng ký một tài khoản GitHub và tạo một Repository (GitHub Repository).

Cài đặt GitHub Desktop, một công cụ trực quan quản lý Local Repository (Kho chứa dữ liệu địa phương).

Cấu hình để có thể đồng bộ hóa dữ liệu bằng GitHub Desktop lên Repository server.

Bước 1:  Đăng ký tài khoản bạn vào trang https://github.com/ 

Bạn chỉ cần nhập username/password và địa chỉ email. Sau khi đăng ký xong bạn cần vào Email kích hoạt tài khoản.

Bước 2: Tạo GitHub Repository. 

Khi đăng ký xong tài khoản và đăng nhập thì bạn bắt đầu tạo

Tạo xong Repository   ra cửa sổ window gõ github desktop và run

Bước 3: Kết nối GitHub và GitHub Desktop

Cách 1 đẩy lên github bằng github desktop

Tạo 1 thư mục rỗng ở ổ bất kỳ

Chọn clone tự động file sẽ được đẩy lên github.

Dòng chữ màu hồng mà hiển thị trên đầu có nghĩa là repository đã tồn tại và được đẩy lên rồi.

Copy 1 file dữ liệu bất kỳ vào  sau đó commit -> push

Khi ta them file có tên là text như hình trên thì  trong github sẽ

Sau khi commit xong ta click vào Publish branch  để push dữ liệu lên github.

Vào github.com ta thấy file đã được đẩy lên thành công

Cách 2:  Đẩy dữ liệu lên github bằng câu lệnh.

Mở file cần đẩy lên github và click chuột phải -> click Git Bash Here sẽ hiển thị ra cửa sổ

  • Đầu tiên ta  khởi tạo bằng câu lệnh

 git init

  • Kiểm tra trạng thái của file bằng lệnh:

          git status

  • Kết nối với tài khoản github bằng 2 dòng lệnh:

git config --global user.name = “daothihang”

git config --global email.name = “daothuhang0696@gmail.com”

Trước khi đẩy file lên github ta phải add và commit lần lượt bằng câu lệnh:

git add * và dùng lệnh git commit -m “tên commit tự đặt” hoặc có thể thay hai lệnh trên bằng 1 lệnh git commit -a -m “tên commit tự đặt”

  • Tiếp theo là đẩy lên github bằng lệnh như sau:

            git remote add origin https://github.com/daothihang/DeMoGit..git
           git push -u origin master

Đã push dữ liệu thành công lên git.

khi làm việc với nhiều người trước tiên ta cần vào repository  chứa project của nhóm và chọn setting -> collaborators  điền tên tài khoản github của người làm cùng project và click vào add collaborator . Người được add vào sẽ vào gmail đăng ký tài khoản accept invite 

Người vừa thêm vào để lấy project về làm thì chỉ cần ra màn hình desktop hoặc ổ đĩa nào đó click chuột phải chọn git bash here và gõ dòng lệnh này để lấy project về làm.

Khi 1 trong hai người đã đẩy project có chỉnh sửa lên mà người còn lại muốn lấy thay đổi đó về làm tiếp ta dùng câu lệnh git pull và làm việc cùng nhau bằng các dòng lệnh trên.
