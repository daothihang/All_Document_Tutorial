Run Project For Automation

I :  Điều kiện môi trường

  • Đã cài Java_Home
  • Cài Maven_Home
  • Cài Appium
  • Máy thật chạy hệ điều hành Android 4.0 trở lên và tải app Slack về máy và cài đặt

II :  Chạy project

  • Bước 1 : Mở project cần chạy

Sau đó ta chọn file pom và lên Previous Locations gõ cmd ->enter nó sẽ hiện ra cửa sổ Command  

Tiếp theo kiểm tra cài đặt maven bằng lệnh mvn install

  • Bước 2: Kết nối với máy thật để chạy dùng dây cap nối giữa máy tính và điện thoại

Note điện thoại vào setting ->tùy chọn nhà phát triển -> tích vào gỡ lỗi USB

dùng câu lệnh adb devices gõ trong command để kiểm tra kết nối với máy và lấy id của máy

Khi kết nối với điện thoại ok ta vào project và mở file appium-android bằng notepad++ hoặc sublime text và chuyền vào id của điện thoại vừa kết nối ->lưu lại và chạy.

Ta quay lại command và gõ lệnh mvn install -Padr    ->enter

Trên điện thoại chạy hiển thị ra màn hình chạy app Slack và bắt đầu test là OK

Chúc các bạn thành công!
