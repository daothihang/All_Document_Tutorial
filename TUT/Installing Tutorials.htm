2.       Setup Android SDK

2.1.   Installing Android SDK

*** System requirements (minimum): 

- 2GB Ram.

- Microsoft® Windows® 8/7/Vista/2003 (32 or 64-bit) .

- 20 -> 30 GB space.

v  Download SDK

- Go to : http://developer.android.com/sdk/index.html

- Go to your file location to install: android-studio-ide-181.5014246-windows.exe

2.2.   Set ANDROID_HOME & Path Environment Variables

v  Right click on My computer

v  Choose Advanced system settings

v  In Advanced tab choose Environment Variable

·         Set ANDROID_HOME

-       On the upper table press New button

-       Variable name: ANDROID_HOME

-       Variable value: C:\Users\lynx1\AppData\Local\Android\Sdk (Directory to your SDK location)

-       Press Ok to save

·         Set Path Environment Variables

-       On the lower table press New button

-       Variable name: Path

-       Variable value:

+ add SDK/tools: C:\Users\lynx1\AppData\Local\Android\Sdk\toolss

+ add SDK/platform-tools: C:\Users\lynx1\AppData\Local\Android\Sdk\platform-tools

-       Press OK to save

3.       Setup Appium

3.1.   Installing Microsoft.Net Framework

v  To install Microsoft.net framework, go to google.com site and search with the key words download latest version of Microsoft.net framework

v  Choose the version you want and click to download.

v  Open the .exe file and install it with the default configuration (eg: NDP472-KB4054531-Web.exe).

3.2.   Installing Node JS

v  Go to https://nodejs.org/en/download/

v  Choose the version that can run on your windows and click to download.

v  Open the .msi file and install it with the default configuration (eg: node-v0.12.7-x86.msi).

3.3.   Installing Appium

v  Go to http://appium.io/

v  Click on the download button to go to the download site.

v  On the following site, find and download the appium-desktop-setup-1.7.1.exe

v  Run the downloaded file to install Appium.

v  Choose For only me then install.
