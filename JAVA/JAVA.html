THIẾT LẬP MÔI TRƯỜNG JAVA_HOME

Điều kiện cài đặt:

  • Eclipse IDE
  • JDK

Các bước cài đặt:

  • Bước 1:Download Eclipse IDE giải nén và cài đặt

        Link download: https://www.eclipse.org/downloads/packages/

  • Bước 2: Download JAVA JDK về máy, tùy thuộc vào win 32 hay win 64 để tải JDK tương ứng về máy.

Giải nén và cài đặt khi cài đặt bạn nên chú ý đến ổ đĩa bạn cài đặt.

Link download: https://www.oracle.com/technetwork/java/javase/downloads/index.html

  • Bước 3: Thiết lập môi trường JAVA_HOME

Mở cửa sổ Window gõ Environment Variables  ->Environment Variables...

Chọn New ->điền JAVA_HOME vào mục Variable Name  và đường dẫn khi bạn cài đặt JDK . Ở đây mình cài ở  D:\Program Files\Java\jdk1.8.0_151 thì mình sẽ copy đường dẫn này và paste vào Variable Value còn bạn nào cài ổ C thì đường dẫn là

C:\Program Files\Java\jdk1.8.0_151   -> Click OK.

Phần System Variables  chọn -> path -> Edit sẽ mở ra 1 cửa sổ mới.

Chọn New -> Browse..  copy đường dẫn cài JDK  paste vào (làm 2 lần)

Note:

      Lần 1 đường dẫn lấy đến thư mục bin , đường dẫn của máy mình là

      D:\Program Files\Java\jdk1.8.0_151\bin

      Lần 2 đường dẫn lấy tới thư mục jre\bin, đường dẫn máy mình là

      D:\Program Files\Java\jdk1.8.0_151\jre\bin 

Bạn cài ở đâu lấy đường dẫn ở đó

Sau đó chọn OK 

Bước 4: Kiểm tra

Vào cửa sổ Window gõ Command Prompt  ->Enter ->gõ lệnh  java -version ->enter

THIẾT LẬP MÔI TRƯỜNG MAVEN_HOME

Điều kiện cài đặt

  • JDK 1.7
  • Maven 3.2.2
  • Windows 8

        Note: Phiên bản Maven 3.2 yêu cầu cài đặt JDK 1.6 hoặc cao hơn, trong khi Maven

                      3.0 / 3.1chỉ cần JDK 1.5 hoặc cao hơn.

   

        Bước 1: Tải Maven giải nén chú ý nhớ đường dẫn nơi giải nén

        Link download: https://maven.apache.org/download.cgi

        Bước 2: Mở cửa sổ Window gõ Environment Variables -> enter -> Environment

  Variables -> System Variables -> New

Lúc này nó sẽ mở ra 1 cửa sổ mới  tại Variable name gõ M2_HOME  ở ô Variable

value  là đường dẫn tới file giải nén  ở đây mình để file đó ở ổ D thì đường dẫn là

D:\apache-maven-3.5.3  và  click -> OK

Vào New  lần 2  nữa và tại ô Variable name gõ MAVEN_HOME , ở ô Variable value

vẫn là đường dẫn file giải nén giống lần 1  click ->OK.

  • Bước 3: Phần System Variables  chọn -> path -> Edit sẽ mở ra 1 cửa sổ mới.

Khi hiển thị ra cửa sổ mới chọn New -> Browse và paste đường dẫn tới file giải nén

Note:   lấy đến thư mục chứa bin , đường dẫn của máy mình là D:\apache-maven-3.5.3\bin sau đó click -> OK.

  • Bước 4: Kiểm tra  mở cửa sổ window gõ Command prompt -> enter ->gõ

 mvn -version nếu hiện ra thông số như hình khoanh đỏ thì bạn đã thiết lập thành công
